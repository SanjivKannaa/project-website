Post-Quantum Network Scanner

Summer Internship at NTU Singapore

Exploring the future of cybersecurity with post-quantum cryptography.

Introduction

The Post-Quantum Network Scanner is an innovative project aimed at enhancing cybersecurity by leveraging post-quantum cryptographic algorithms. This project was undertaken during my summer internship at NTU Singapore, focusing on the challenges posed by quantum computing to current cryptographic systems. The project's goal is to develop a network scanner that can capture and analyze network traffic, assessing the security of cryptographic algorithms in use and their readiness for a post-quantum world.

Objectives

The primary objectives of the Post-Quantum Network Scanner project are:

  • To develop a tool that can scan network traffic and identify the cryptographic algorithms being used.
  • To evaluate the security of these algorithms in the context of post-quantum computing.
  • To provide insights and recommendations for organizations on transitioning to post-quantum cryptography.

Technologies Used

  • Python
  • Post-Quantum Wireshark
  • Electron JS
  • Docker

Methodology

The project was conducted using a structured approach involving the following steps:

  • Research: Comprehensive research was conducted on existing cryptographic algorithms and the impact of quantum computing on their security.
  • Design: A network scanner was designed to be used as a centralized system for an enterprise/campus network.
  • Implementation: Authorization and user management were integrated into the network scanner.
  • Testing: Extensive testing was carried out to evaluate the performance and security of the application.
  • Analysis: The results were analyzed to provide insights into the readiness of current cryptographic systems for a post-quantum era.

Conclusion

The Post-Quantum Network Scanner project highlights the critical need for organizations to prepare for the challenges posed by quantum computing. By adopting post-quantum cryptographic algorithms, businesses can protect sensitive data and ensure secure communications in a future where quantum computers may compromise current encryption methods.
